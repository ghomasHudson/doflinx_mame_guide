# DOFLinx MAME File Guide

This site explains how to write and maintain DOFLinx `.MAME` files to enable in-game effects.

Use the [Documentation](docs/documentation.md) for task-based guidance and the [API Reference](docs/api-reference.md) for quick lookup tables.

If you make any improvements to `.MAME` files, please submit them to the DOFLinx GitHub repository so the community can all benefit: [DOFLinx MAMEV3](https://github.com/DOFLinx/DOFLinx/tree/main/MAMEV3).
